## 🚀 快速开始

### 第一步：安装应用
按照上面的安装方法完成应用安装。

### 第二步：授予权限
首次启动时，应用会请求访问照片的权限：
1. 点击"授予权限"按钮
2. 在系统弹窗中选择"允许"
3. 应用将自动扫描设备上的所有照片

> 💡 **提示**：不同 Android 版本权限提示可能略有差异，请根据实际提示操作。

### 第三步：开始欣赏
- 照片加载完成后，会自动开始播放（如已开启自动播放）
- 使用遥控器方向键浏览照片
- 点击确认键可随时暂停/继续

## ⚙️ 设置说明

按遥控器**菜单键**或点击屏幕**设置按钮**打开设置界面。

### 切换间隔
调节照片自动切换的时间间隔：
- 点击 **+** 增加间隔
- 点击 **-** 减少间隔
- 范围：1-30 秒
- 默认：3 秒

### 动画效果
选择您喜欢的切换动画：
- 从下拉列表选择具体动画类型
- 或开启"随机动画"，每次切换都有不同效果

### 显示选项
- **显示指示器** - 显示/隐藏底部位置指示器
- **自动播放** - 启动时是否自动开始播放

### 快捷操作
- **重置** - 一键恢复默认设置
- **应用** - 保存当前设置并关闭对话框

## 💡 使用技巧

### 技巧 1：选择合适的切换间隔
- **风景照片**：建议 5-10 秒，让您有足够时间欣赏细节
- **人物照片**：建议 3-5 秒，节奏更轻快
- **快速浏览**：设置 1-2 秒，快速查看大量照片

### 技巧 2：动画效果搭配
- **正式场合**：使用淡入淡出，简洁大方
- **家庭聚会**：启用随机动画，增添趣味
- **展示作品**：使用缩放效果，突出视觉冲击

### 技巧 3：遥控器快捷操作
- 看到喜欢的照片？按确认键暂停，细细欣赏
- 快速跳过照片？连续按方向键快速切换
- 想调整设置？直接按菜单键，无需退出播放

## ❓ 常见问题

### Q1：应用显示"未找到照片"怎么办？
**解决方法**：
1. 确认设备上确实有照片文件
2. 检查是否已授予存储权限
3. 尝试重启应用
4. 如果照片在 SD 卡上，确保 SD 卡已正确挂载

### Q2：为什么有些照片显示不出来？
**可能原因**：
- 照片格式不支持（支持 JPG、PNG、WEBP 等常见格式）
- 照片文件损坏
- 照片体积过大（建议单张不超过 10MB）

### Q3：动画播放卡顿怎么办？
**优化建议**：
1. 增加切换间隔时间
2. 减少照片分辨率（推荐 1920x1080）
3. 关闭其他后台应用
4. 重启电视设备

### Q4：遥控器按键没有反应？
**检查步骤**：
1. 确认设备是 Android TV 系统
2. 更换遥控器电池
3. 重新配对遥控器
4. 重启应用或设备

### Q5：如何卸载应用？
在电视系统设置中：
1. 进入"应用管理"
2. 找到"FlowAlbum"
3. 选择"卸载"
